# SympTrack AI

SympTrack AI is a machine learning web application that predicts possible diseases based on user-selected symptoms. The system uses machine learning models to analyze symptom patterns and provide real-time predictions through an interactive Streamlit interface.

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
- NumPy

## Features
- Disease prediction using Random Forest classifier
- Interactive symptom selection interface
- Real-time disease prediction
- Confidence score for predictions
- Simple and user-friendly web interface
